# LawFinder

LawFinder is a simple legal search aggregator designed to query and compile results from multiple legal databases simultaneously.

---

## Features

* **Parallel Search:** Utilizes Python's `asyncio` and `ThreadPoolExecutor` to query multiple sources in parallel.


* **Automated Browsing:** Employs headless Selenium with Firefox to configure and submit advanced search queries.


* **Clean UI:** A single-page dashboard built with Tailwind CSS that displays organized results side-by-side.



---

## Supported Portals

* **EUR-Lex:** Official repository of European Union law.


* **N-Lex:** Gateway to national law websites of EU countries.


* **Legislationline:** Database of international and domestic legislation.



---

## Tech Stack

* **Back-end:** Python, Flask, Asyncio, Selenium


* **Front-end:** HTML5, Tailwind CSS, Vanilla JavaScript
